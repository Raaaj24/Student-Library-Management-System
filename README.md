# Student Library Management System

This repository contains the source code and project files for the Student Library Management System, developed as part of an academic project.

## Project Overview
A Spring Boot backend application for managing books, members, borrowing, and returns.

## Technologies Used
- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL
- Maven

## Purpose
Demonstrates REST APIs, JPA relationships, exception handling, pagination, sorting, and JPQL.
